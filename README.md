# Fundamentos de Python

Repositório com exercícios desenvolvidos para praticar conceitos básicos e lógica de programação utilizando Python.

## Exercícios

### 1. Verificação de Tipos
Programa que identifica o tipo de dado das entradas informadas pelo usuário.

### 2. Cálculo de Aluguel de Carro
Programa que calcula o valor total de um aluguel de carro com base nos dias utilizados e quilômetros percorridos.

### 3. Cálculo de Aumento Salarial
Programa que calcula o valor do aumento salarial e o novo salário de um funcionário.

## Assuntos praticados

- Variáveis
- Tipos de dados
- Entrada e saída de dados
- Operações matemáticas
- Porcentagem
- Conversão de tipos
- Formatação de números
- Lógica de programação
